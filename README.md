# AI-Powered Attendance Automation Prototype

## Overview

This project is an AI-powered attendance automation system developed as part of the SafeX Solutions Business Automation Research internship.

The system automates employee attendance using QR Code verification and geofencing while providing AI-generated attendance insights for managers.

## Features

- QR Code Attendance
- Geofence Validation
- SQLite Database
- Flask Backend
- Pandas Data Processing
- Gemini AI Attendance Analysis
- Manager Dashboard

## Technologies

- Python
- Flask
- SQLite
- Pandas
- Google Gemini API
- Git & GitHub

## Project Structure

- Employee Authentication
- Attendance Validation
- Attendance Storage
- AI Attendance Analysis
- Dashboard Reporting
